BigO

According to quiz.
Constant Time: Given an input of size n, it only takes a single step for the algorithm to accomplish the task.
Value look ups when you know the key (objects) or the index (arrays) always take one step, and are thus constant time.
Logarithmic time: given an input of size n, the number of steps it takes to accomplish the task are decreased by some factor with each step.
If you know which side of the array to look on for an item, you save time by cutting out the other half.
Linear Time: Given an input of size n, the number of of steps required is directly related (1 to 1)
Single for loops are almost always linear time.
ou have to look at every item in the array or list to accomplish the task.
Quadratic Time: Given an input of size n, the number of steps it takes to accomplish a task is square of n.
Exponential Time: Given an input of size n, the number of steps it takes to accomplish a task is a constant to the n power (pretty large number).
O(n!)>O(2^n)>O(n^2)>O(n log n)>O(n)>O(log n), O(1)
